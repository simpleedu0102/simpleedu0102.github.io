Write it! 每日一写 | DAY 0003：材料练习 3

Posted on 2024-11-14 In Write it! 每日一写 Views: Disqus:

Write it! 每日一写 | DAY0003：材料练习 3

同学们好，欢迎来到 Write it! 每日一写课堂。今天要写的内容，依然是《怎样成为品学兼优的学生》这篇说明文。让我们只聚焦在一个重点，那就是 “怎样想到作文材料呢？” 然后，写出一段 60 个字左右的文本吧！

在一开始写这篇文章时，我们已经拆解了作文题目的要求，了解到题目能够拆分成 “怎样” 和 “成为品学兼优的学生” 两个部分。

因此，我们能够确定的是，我们要写的这篇作文是，五个段落中，需要至少回答 3 个 “方法”。什么东西的方法呢？那就是题目的第二个部分：“成为品学兼优的学生”。

从题目看来，这篇作文要写是：

  • 一、说明文；
  • 二、答案是 “方法”；
  • 三、一个学生行为道德变好的方法 + 一个学生学习成绩变好的方法（第三个答案，二选一）；
  • 四、没有格式要求（演讲稿、信函……）

接下来，让我们一起来想想 “答案三” 是什么，然后用 “答案三” 来填满这段文本的空格吧！在进入 “学业成绩” 的答案之前，还记得老师在上一堂课说的，我们需要学习多种变化的段落写法，才能应对不同的句子衔接吗？所以，我们今天依然是写道德价值观，不过，段落的模板有了一些变化。

XX 第三，答案三。答案三是很重要的品德。我们应该做到答案三，这是因为，一个运用答案三的例子。 因此，具备答案三是一个学生必备的基本品质。

在填写答案三时，让我们把前一堂课的道德价值观和例子套入新的段落模板，并且顺便复习想出例子的四个小技巧吧！

【例子来源】

  1. 从过去读过的任何材料找例子（古代的人、看过的故事、认识的名人……）；
    • 乐于助人的性格：（★ 【Simply Speak 极简朗读】的材料）当我们在路上看到一位老奶奶拄着拐杖，步履蹒跚地过马路时，我们可以上前扶住她，慢慢地将她送到对面。
    • 勇敢的品格：（★ 【Simply Speak 极简朗读】的材料）我们可以战胜内心的恐惧，勇敢地挑战校内校外的演讲比赛或辩论比赛，可以帮助我们建立自信心、提升口才。这也可能将为我们带来更多意想不到的机会。
  2. 从自己日常生活的经验找例子；
    • 乐于助人的性格：当同学忘记带食物和零用钱到学校时，如果我们主动和对方分享我们的食物，或是借对方一些钱，能够避免对方挨饿，影响他的学习效率。这么一来，在我们需要帮助时，别人也会更愿意帮助我们。
    • 勇敢的品格：在面对困难，或是新事物时，只有用勇气去面对，才能够解决问题，学习到新的知识、技能或是经验。毕竟，逃避是没有用的。勇敢而主动，也有助我们扩大朋友圈，拓展人际关系。
  3. 从相反的角度找例子；
    • ★★ 乐于助人的性格：如果我们不愿意在别人有难时伸出援手，给予援助，那么当我们自己身处困境时，也很容易落得孤立无援（alone and unsupported/helpless）的下场。
    • 勇敢的品格：在面对困难，或是新事物时，一个胆小怕事、只会逃避的人，是永远无法走出困境的，因此所以，这么做会让自己一直原地踏步，浪费生命。
  4. 从其他人的角度找例子。
    • 乐于助人的性格：其他人若是觉得一个人既不善良，也无爱心的话，就会不想和他做朋友、不想帮助他，甚至不愿意和他交谈。
    • 勇敢的品格：其他人若是觉得一个人懦弱无能，面对事情只会逃避，可能会因此看不起他，甚至欺负他，让他的处境恶化 / 雪上加霜（become worse）。

好了，以上就是老师以 “乐于助人的性格” 或 “勇敢的品格” 作为 “答案三” 填入空格之后的八个例子，供你参考。那么，你的答案，是什么呢？在这三节课中，我们使用的段落模板是最容易和基本的。这对基础不稳的同学有用，但是对想拿高分的同学，可能没有效果。希望同学们能够在老师的陪伴下，努力升级打怪，变得越来越厉害，那么，老师在后续课程，就能带领同学们一起解锁更多种段落句子衔接的写法，包括更加出彩的写法了。下一节课，我们将进入学业成绩的答案这个部分，同学们可以期待一下。

好的，今天的课先讲到这里，恭喜你又进步啦！棒棒的你，快到作业区提交作业吧！我们下节课见！
