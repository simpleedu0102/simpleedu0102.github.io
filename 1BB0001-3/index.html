Write it! 每日一写 | DAY 0010：结构练习 3

Posted on 2024-11-15 Edited on 2024-11-16 In Write it! 每日一写 Views: Disqus:

Write it! 每日一写 | DAY0010：结构练习 3

同学们好，欢迎来到 Write it! 每日一写课堂。今天要写的内容，依然是《谈培养阅读的兴趣》这篇说明文。让我们只聚焦在一个重点，那就是 “怎样建立作文结构呢？” 然后，写出一段 60 个字左右的文本吧！

在一开始写这篇文章时，我们已经拆解了作文题目的要求，了解到题目能够拆分成 “谈” 和 “培养阅读的兴趣” 两个部分。并且，我们先把要 “谈” 的方向，定在比较容易回答的 “方法” 上。

因此，我们能够确定的是，我们要写的这篇作文是，五个段落中，需要至少回答 3 个 “方法”。什么东西的方法呢？那就是题目的第二个部分：“培养阅读的兴趣”。简单来说，就是 “让人喜欢上阅读” 的意思。

从题目看来，这篇作文要写是：

  • 一、说明文；
  • 二、答案是 “方法”；
  • 三、三个让人喜欢上阅读的方法；
  • 四、没有格式要求（演讲稿、信函……）。

接下来，我们继续来复习前面学习到的两个技巧吧！请同学们抓紧上车，系好安全带，我们出发吧！

还记得小简老师前面教给大家的第二个技巧吗？那就是，在写说明文的 “方法” 时，我们可以尝试从 “小到大” 或是 “大到小” 的角度去展开。“从小到大” 的意思是，同学可以从 “自己” 能做出什么方法，想到 “父母” 或 “家庭” 、“学校” 、“社区” 、“媒体” 、“政府” 等等，大家又可以做出什么方法呢？这样一想，同学就能够得出一些答案哦！你也可以反过来写 “大到小” ，“政府” 、“媒体” 、“社区” 、“学校” 、“父母” 或 “家庭”，到 “自己” 能够做什么，来写出 “方法” 的答案。

上一节课，我们用了 “父母” 、“学校” 作为 “答案一” 和 “答案二” ，现在，在进一步放大范围之前，我们来写个特别的，或许我们可以把家庭和学校融合在一起写？这是不是挺有趣的呢？因为家庭和学校，都是同学们最熟悉的写作材料。不知道同学们还记不记得，在最初第一堂课，小简老师给同学们讲过，想出作文材料的四个来源，其中一个就是 “从自己日常生活的经验去找” 呢？

接下来，我们把 “家庭” 和 “学校” 一起作为 “答案三” ，套用到 “A Cat May Only Sleep Extra” 这个模板，帮助大家来复习一下这个口诀和使用方法吧！

第一个技巧，是教你怎么写说明文的 “答案” 这个段落。假设这个段落要写五个句子，我们可以这样安排：

第一句：答案（Answer）

第二句：原由（Cause）

第三句：做法（Method）

第四句：结果（Outcome）

第五句：小结（Summary）

其他随意：例子（Example）

要记得上述说明文内容段落的写作顺序，老师给大家编了三个口诀，同学们可以按照自己的喜好，选择自己容易记得的口诀。不过，为了方便教学，老师未来在课堂上会固定使用第一句口诀：

【口诀】

  1. A Cat May Only Sleep Extra.
  2. A Cat May Order Spicy Egg.
  3. A Cat May Own Stripy Ears.

我们来分析一下这个例子，可以发现：

XX 最后，在家里或学校营造浓郁的读书氛围，也能激发阅读的兴趣。比如，在家里的客厅，或者在学校班上设立 “阅读角落”，大家一起来阅读。在校园中拉起 “爱书吧，它是知识的源泉！” 的横幅、在橱窗中贴出 “新书推荐” 的海报等等，也可以加强读书氛围。此外，把布置漂亮的新书展览（display）于图书室，并组织同学们前去参观，同学们一定会被那色彩缤纷的封面和插图、妙趣横生的内容深深吸引。所以说，新加坡的师长们花心思在家庭和校园中打造一起阅读的风气，就能让新加坡成为一个书香国家。

【Answer】最后，在家里或学校营造浓郁的读书氛围，也能激发阅读的兴趣。【Method / Example】比如，在家里的客厅，或者在学校班上设立 “阅读角落”，大家一起来阅读。【Method / Example】在校园中拉起 “爱书吧，它是知识的源泉！” 的横幅、在橱窗中贴出 “新书推荐” 的海报等等，也可以加强读书氛围。【Method / Example】此外，把布置漂亮的新书展览（display）于图书室，并组织同学们前去参观，【Outcome】同学们一定会被那色彩缤纷的封面和插图、妙趣横生的内容深深吸引。【Summary】所以说，新加坡的师长们花心思在家庭和校园中打造一起阅读的风气，就能让新加坡成为一个书香国家。

到这里为止，大家能感觉到第一个技巧，是怎样帮助同学们又写出一段说明文内容了吗？

好了，以上就是老师以 “家庭” 和 “学校” 作为 “答案三” 套用 “A Cat May Only Sleep Extra” 这个模板的例子，供你参考。那么，你的答案，是什么呢？

好的，今天的课先讲到这里，恭喜你又进步啦！棒棒的你，快到作业区提交作业吧！我们下节课见！
